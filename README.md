## Overview
A high-end, Awwwards-tier personal portfolio architected from the ground up to showcase specialized digital services including Full-Stack Web Development, App Engineering, and Website Transformation. 

Designed with absolute mobile-first precision, relentless CSS fluid grids, and zero-latency physics-based micro-interactions.

## ⚡ Core Architecture & Engineering

- **Abstract Physics Rendering**: Real-time ambient Aurora orbs and DOM-calculating custom pointer rings driven by `mix-blend-mode` math and absolute strict CSS rendering.
- **Spring-Engine Modals**: Immersive 100vh curtain drop menus and overlapping floating project modals natively orchestrated using `framer-motion` physics-based trajectories.
- **Mobile-Fluid Grid Mathematics**: Rigorous CSS `clamp()` logic perfectly scaling all typography and structural Padding constraints infinitely across 4K displays straight down to 320px Apple viewports. No hard-snapping breakpoints.
- **Smooth Scroll Integration**: Executed with `@studio-freight/lenis` handling requestAnimationFrame loops to intercept the native scrollbar, completely removing jitter.

## 🛠️ Technology Stack
- **Framework**: React 18 + Vite HMR Env
- **Styling**: Pure Modular CSS (Zero-framework dependencies, pure architectural specificity)
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **Scroll Engine**: Lenis Scrolling

## 🚀 Running Locally

1. Clone the repository:
```bash
git clone https://github.com/NishadRaval/Portfolio.git
```

2. Install the necessary dependencies:
```bash
npm install
```

3. Spin up the Vite development server:
```bash
npm run dev
```
